# Удаление своих последних сообщений в Телеграмме
Скрипт удаляет сообщения за последние 10 (число меняется в настройках) дней во всех группах, исключая личные сообщения. Необходимо получить ID и ID_HASH телеграмма здесь https://my.telegram.org/auth?to=apps и заполнить .env. В первый раз придётся вводить номер телефона аккаунта и проверочный код. Не забудьте, что лучше всего создать локальную среду через python3 -m venv .venv, source ..., pip install... и deactivate. 
Также можно исключить некоторые группы (не удалять сообщения), указав их в .env (обязательно проверяйте ID, он не всегда совпадает с информацией о группе телеграмм, также смотрите логи).
